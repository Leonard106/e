# TE_2026_TRAIDORES_PROYECTO_FINAL
🏫 Sistema de Control de Horas de Servicio Social
Laboratorio de Macrodatos — FES Aragón, UNAM

📌 Descripción del Proyecto
Sistema web para administrar y controlar las horas de servicio social de alumnos asignados a programas del Laboratorio de Macrodatos de la FES Aragón, UNAM.

🛠️ Tecnologías

Backend: Java, Spring Boot, Spring Data JPA, API REST
Frontend: Spring Web MVC, Thymeleaf
Base de datos: MySQL
Infraestructura: Docker, Docker Compose
Control de versiones: Git & GitHub
Gestión del proyecto: SCRUM + Trello
